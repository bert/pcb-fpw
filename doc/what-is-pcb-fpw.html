What is PCB FPW ?

PCB FPW is a FootPrintWizard for PCB.

fpw (FootPrintWizard) is a CLI based program for the creation of footprint files used by the pcb layout editor.
pcb-gfpw (gtkfootprintwizard) is a GTK based version (eye-candy).

There is a User Manual for pcb-gfpw.

What is PCB ?

PCB is an interactive Printed Circuit Board layout editor for the X11 window system.
PCB includes a rats nest feature, design rule checking, and can provide industry standard RS-274-X (Gerber), NC drill, and centroid data (X-Y data) output for use in the board fabrication and assembly process.
PCB offers high end features such as an autorouter and trace optimizer which can tremendously reduce layout time. For more information see http://pcb.gpleda.org.
